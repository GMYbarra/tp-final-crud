TP Final - Sistema de Pedidos

Este sistema permite gestionar productos, clientes y pedidos usando una API REST en .NET y una base de datos MySQL.

Secciones principales

  • Productos: alta, baja, modificación y listado de productos.
  • Clientes: (opcional) administración de clientes.
  • Pedidos: (opcional) creación y consulta de pedidos.
